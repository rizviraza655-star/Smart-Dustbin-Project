# Smart Dustbin 🚮

An automatic, touchless waste management system built using Arduino.

Description
The Smart Dustbin is designed to improve hygiene by eliminating the need to touch the bin lid. It uses an ultrasonic sensor to detect hand movement and automatically opens the lid using a servo motor.

⚙️ Components Used
- Arduino Uno  
- HC-SR04 Ultrasonic Sensor  
- SG90 Servo Motor  
- Breadboard & Jumper Wires  
- Power Supply (5V)

🔄 Working Principle
- The ultrasonic sensor detects an object (hand) within a certain distance (~20 cm).
- Arduino processes the signal.
- Servo motor rotates to open the lid.
- After a few seconds, the lid closes automatically.

🎯 Features
- Touchless operation  
- Improves hygiene  
- Simple and cost-effective  
- Easy to build
